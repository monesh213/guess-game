🎯 Guess The Number Game (1 to 10)

A fun and interactive Guess The Number game built using HTML, CSS, and JavaScript.
The user must guess a randomly generated number between 1 and 10. The player starts with 10 points, and each wrong guess decreases the score by 1.

🚀 Live Game Features

✅ Random number generated between 1 - 10
✅ User input using number box
✅ Stylish UI with modern card design
✅ Gradient background design
✅ Displays result message (RIGHT / WRONG)
✅ Score system starting from 10
✅ Score decreases on wrong attempts
✅ Game Over when score reaches 0
✅ Alerts user when they win

🛠️ Technologies Used
HTML (Structure)
CSS (Design & Styling)
JavaScript (Game Logic + DOM Manipulation)
